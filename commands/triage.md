# Pediatric Symptom Triage

Run a structured symptom assessment for a child. Apply **pediatric-safety** and **symptom-assessment** skills throughout.

## Intake

Ask for any missing details:

1. Child's **age** (months if under 2 years)
2. **Primary symptom** and when it started
3. **Fever** -- temperature and method if known
4. **Severity** -- eating, drinking, activity, alertness
5. **Other symptoms**
6. **Country** -- for emergency numbers and care access

If red flags are reported at any point, **stop** and direct to emergency services.

## Assessment Flow

1. **Red-flag screen** -- check emergency criteria from pediatric-safety skill
2. **Triage classification** -- Emergency / Urgent / Non-urgent / Self-care
3. **Educational summary** -- age-appropriate, non-diagnostic explanation
4. **Action plan** -- home care (if appropriate), when to call doctor, when to go to ER
5. **Disclaimer** -- not medical advice; consult pediatrician

## Output Format

```
## Triage Result: [LEVEL]

### Summary
- **Age**: ...
- **Symptoms**: ...
- **Duration**: ...

### What This Could Mean
[Non-diagnostic explanation]

### Recommended Next Steps
1. ...

### Seek Emergency Care Immediately If
- ...

### Contact Your Pediatrician Today If
- ...

---
*Mamabora provides general health education, not medical diagnosis or treatment.*
```

## Follow-Up

If triage level is Non-urgent or Self-care, offer to:

- Explain comfort measures in more detail
- Help prepare questions for the pediatrician visit
- Set a check-in timeline ("If not better in 48 hours, call your doctor")
